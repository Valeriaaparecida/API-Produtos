# CRUD de Produtos - Spring Boot + JPA + H2

Este é um projeto educativo que implementa um **CRUD (Create, Read, Update, Delete)** de produtos usando **Spring Boot**, **Spring Data JPA** e banco de dados em memória **H2**.

O objetivo é demonstrar como criar, ler, atualizar e deletar registros em um banco usando **ORM** (JPA/Hibernate) sem escrever SQL manualmente.

---

## 🛠 Tecnologias Utilizadas

- Java 21  
- Spring Boot 4.0.3  
- Spring Data JPA  
- H2 Database (em memória)  
- Maven  
- Lombok (opcional, para reduzir boilerplate)
